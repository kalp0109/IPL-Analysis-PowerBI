*IPL Analysis Dashboard (2008–2025)*

## Project Overview

This Power BI project provides a complete analysis of the Indian Premier League (IPL) from 2008 to 2025 using interactive dashboards and visual analytics. 
The dashboard delivers insights into team performance, player statistics, season-wise trends, and tournament achievements.
The project focuses on transforming raw IPL data into meaningful business-style sports analytics dashboards.

## Key Features
Season-wise IPL analysis (2008–2025)
IPL Winners and Runner-Up analysis
Orange Cap and Purple Cap statistics
Points Table analysis
Total Fours and Sixes analysis
Team performance comparison
Player statistics and trends
Interactive filters and slicers
Dynamic KPI cards and visuals

## Tools & Technologies Used
Microsoft Power BI
Power Query
DAX
Data Modeling
Data Visualization

## Insights Covered
Most successful IPL teams
Highest run scorers across seasons
Top wicket takers
Team-wise win analysis
Boundary and six hitting trends
Season performance comparison
Points table standings and qualification trends
